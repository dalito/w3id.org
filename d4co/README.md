# /d4co

This W3ID provides a persistent URI namespace for the [Design-for-Circularity Ontology](https://gitlab.com/dlr-dw/d4co) developed as part of the [Methods and Technologies for an intelligent Circularity of Materials (MaTiC-M) project](https://www.dlr.de/de/dw/forschung-transfer/projekte/matic-m).

## Uses

Provides access to the HTML and RDF representations of the latest version of the ontology.

## Example URLs

* Latest version, whole ontology: <https://w3id.org/d4co/>
* Latest version, specific resource: <https://w3id.org/d4co/D4C_00000456>
